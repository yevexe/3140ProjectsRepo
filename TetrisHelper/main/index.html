TetrisHelper (v0.4.1)

Authors: Yevgeniy, Daniel, Arsen

Jstris is a fast-paced, browser-based Tetris game with a competitive community and a public API that tracks detailed user and gameplay data. This project transforms that data into a personalized analytics tool designed to help players better understand and improve their performance.
